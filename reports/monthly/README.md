# Monthly Reports

This directory contains monthly calibration reports and performance metrics.

## Contents

- **threshold_calibration_YYYY-MM.xlsx** - Monthly threshold recalibration results
- **match_statistics_YYYY-MM.csv** - Aggregated match type distributions
- **synonym_growth_YYYY-MM.csv** - Synonym corpus growth metrics
- **validation_workload_YYYY-MM.csv** - Validation effort tracking

## Report Generation

Reports are automatically generated by:
```bash
python scripts/10_monthly_calibration.py
```

## Retention

Keep reports for minimum 2 years for audit trail purposes.
