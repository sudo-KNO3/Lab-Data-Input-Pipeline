# Daily Reports

This directory contains daily validation session logs and anomaly detection reports.

## Contents

- **validation_session_YYYY-MM-DD.csv** - Daily validation activity
- **new_unknowns_YYYY-MM-DD.csv** - Chemicals requiring manual review
- **match_confidence_dist_YYYY-MM-DD.png** - Daily confidence distribution plots
- **anomalies_YYYY-MM-DD.log** - Flagged edge cases or system issues

## Report Generation

Reports are automatically generated by:
```bash
python scripts/09_ingest_daily_validations.py
```

## Cleanup

Daily reports are typically archived after 30 days unless flagged for review.
